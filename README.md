Memory Blocks Game

A card matching memory game built with vanilla HTML, CSS, and JavaScript.

How to Play


Click "Start Game" and enter your name
Cards will briefly reveal themselves at the start
Click two cards to flip them
If they match, they stay flipped
If they don't match, they flip back
Match all cards before reaching 12 wrong tries to win


Features


Cards preview at the start of the game
Timer tracking how long you take to finish
Wrong tries counter
Performance level based on wrong tries
Win and Game Over popups with stats
Shuffle system so cards are in a different order every game

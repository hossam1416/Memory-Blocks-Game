# Memory Blocks Game

A card matching memory game built with vanilla HTML, CSS, and JavaScript.

## How to Play

1. Click "Start Game" and enter your name
2. Cards will briefly reveal themselves at the start
3. Click two cards to flip them
4. If they match, they stay flipped
5. If they don't match, they flip back
6. Match all cards before reaching 12 wrong tries to win

## Features

- Cards preview at the start of the game
- Timer tracking how long you take to finish
- Wrong tries counter
- Performance level based on wrong tries
- Win and Game Over popups with stats
- Shuffle system so cards are in a different order every game

## Levels

| Wrong Tries | Level |
|-------------|-------|
| 0 – 5 | Excellent |
| 6 – 11 | Good |
| 12+ | Bad |
